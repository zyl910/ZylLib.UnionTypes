# ZylLib.UnionTypes
Union types in C# (C#中的联合体类型)

## Instruction

Union types :

* `UnionShort`: 2 byte union type. Example `short[1]` .
* `UnionInt`: 4 byte union type. Example `short[2]`, `int[1]`, `float[1]` .
* `UnionLong`: 8 byte union type. Example `short[4]`, `int[2]`, `long[1]`, `float[2]`, `double[1]` .
